Ashley's 3 Icons

Icon Development and Process

Message Icon

Application

This message icon is intended to show the user whether the app is open or closed. It will be animated while it is opened and it will stop while it is closed.

Feature

A message icon which notifies the user whether the app is opened or closed. It also has standard messaging features such as a notification when the user sends or recieves a text message.

Interface

This icon is intended to be used with a phone interface. It will appear on the bottom of the main home phone screen and will also be in the general app listing screen.

Play Icon

Application

This play button is intended to show the user when the video has been played or paused. This button informs the user of an action.

Feature

This play button was created using a "bounce" which helps to transition from one state "play" to the next state "pause". This animation only plays when the player switches between the play and pause state.

Interface

This icon is intended to be used with a Youtube or other video software interface. It will appear in the corner of the video and will start with the intial "play" state until the user press it to trasition the state to the "pause" state.

Fig.2 Play Button Icon

Present Icon

Application

This present icon is intended to let the user know if the user turned on the gift option for the item purchase. This icon will be animated when turned on and unanimated when turned off.

Feature

This present icon notifies the user of gift purchase which includes special packaging and a gift return receipt in case the package was damaged.

Interface

This icon is intended to be used with an online store such as Amazon or Ebay. It would be included in the purchasing process and would be an option right next to the regular purchase option.
